FAQ

Perguntas frequentes

01. Por onde eu começo?

Assista aos tutoriais antes de avançar para a criação dos links.

02. Como eu consigo o cookie?

O cookie é obtido quando o alvo loga e insere o código 2FA.

03. O que é Bypass?

Ferramenta para forçar a remoção do e-mail verificado da conta.

04. O que é Auth?

Ativa um autenticador e gera uma chave de acesso para você.

05. O que é Summary?

Fornece os dados de inventário e histórico de Robux.

06. Como usar o Método Link Fake?

Crie o link no painel, envie ao alvo e aguarde os dados no bot.

07. O que é Autosecure?

Sistema que troca os dados da conta automaticamente ao capturar.

08. Meu link caiu, o que eu faço?

Use os domínios secundários disponíveis no painel Splunk X.

09. O que é Dual-Hook?

Envia os resultados para dois webhooks diferentes ao mesmo tempo.

10. Onde recebo ajuda?

Abra um ticket em nosso servidor oficial do Discord.
